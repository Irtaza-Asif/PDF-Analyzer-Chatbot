# 🤖 PDF Analyzer Chatbot

A Retrieval-Augmented Generation (RAG) chatbot that allows users to upload PDFs and ask questions about their content.

## 🚀 Features

* 📄 Upload and analyze PDFs
* 💬 ChatGPT-style interface
* 🧠 Context-aware answers using RAG
* 📚 Source citations (with page numbers)
* ⚡ Fast semantic search with embeddings

## 🛠 Tech Stack

* Python
* Streamlit
* LangChain
* HuggingFace Transformers
* ChromaDB

## 📦 Installation

```bash
git clone https://github.com/Irtaza-Asif/PDF-Analyzer-Chatbot.git
cd PDF-Analyzer-Chatbot

python3.10 -m venv rag_env
source rag_env/bin/activate

pip install -r requirements.txt
```

## ▶️ Run

```bash
python -m streamlit run app.py
```
